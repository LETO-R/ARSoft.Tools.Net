h1. ARSoft.Tools.Net Fork

This is a fork of http://arsofttoolsnet.codeplex.com/ based on the v1.6.0 release.

h2. What is different?

* Minor changes to make ARSoft.Tools.Net compilable using MonoDevelop
* DnsSocksClient class
 * Possible to perform TCP-DNS requests through a SOCKS 4 proxy
